Files Structure

  • /superio-html/ – main template folder with all files to getting started
    • /*.html – all html files
    • /css – all css files
    • /fonts – fonts and icons
    • /images – all images used in template (note, in downloadable archive this is placeholder images)
    • /Js – all js plugins, controllers and scripts files
  • /superio-doc – template documentation

Editing Files

If you don’t know anything about HTML and web pages, before doing anything with this template we recommend you to take at least some basic tutorial or hire somebody experienced to do this.


As for tutorials we recommend https://www.w3schools.com/. Editing php and JavaScript files is really advanced stuff, only experienced person should edit them. If you’re not sure, do not change anything, it might cause serious damage to the scripts. Hiring somebody, who knows what to do, is better solution.


If you are not familiar with these technologies, do not play around with these files.

Show Content

  • Inspect element – when editing html and css files utilities like firebug (free firefox plugin) are great helpers. With them you can easily find element style, even the stylesheet name and line number where the inspected style rule is.
  • Edit html/css/js/php files – use your favorite text editor. Editor, that highlights syntax is good choice, SublimeText (free) or Notepad++ (free) might be enough.
  • View pages – any common browser like Mozilla Firefox (free)
  • Upload on server – look for some FTP program – FileZilla (free)

Installation

To install Superio Template on your web server follow next steps:


  • Download the files from Themeforest and unzip them to the local disk on your computer
  • Inside the extracted folder you will see Two folders named superio-Doc and superio-html
  • Using an FTP Client (such as Filezilla or Total Commander), upload all of the contents of the superio-html folder to your server

Jquery and javascript


This theme imports three types of Javascript files.

  • jQuery
  • Custom scripts (script.js)
  • Some plugins

  • Most of the animation in this site is carried out from the customs scripts. There are a few functions worth looking over.

In addition to the custom scripts, I have implemented few "tried and true" plugins to create the effects. This plugin is packed, so you won't need to manually edit anything in the file. The only necessary thing to know is how to call the method
